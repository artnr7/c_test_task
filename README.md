# c_test_task

Интерпретатор скриптового языка для загрузки shared library с поддержкой 2-ух команд: 
use "lib_name.so"
call "function_name"

Сборка
```
make
```

Запуск
```
make run
```


```
Для проведения тестов установите check.h
pacman -S check
apt install check
```


```
Для проведения памяти установите valgrind
pacman -S valgrind
apt install valgrind
```
